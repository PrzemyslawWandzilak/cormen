# Longest common subsequence
1. **Dependencies**
    - `glib-2.0`
2. **Usage**  
	Compile with `make`  
	Syntax: `./main`  
    - input: two `\n` separated words
    - output: longest common subsequence
3. **Example operation**  
  `» ./1-lcs`  
  `abrakadabra`  
  `yabbadabbadoo`  
  `abadaba`  
---
Tested under kernel 4.16.8-1-ARCH and g++ 8.1.0.
