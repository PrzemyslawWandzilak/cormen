## Longest common subsequence
1. **Usage**  
	Compile with `make`  
	Syntax: `./1-lcs`  
    - input: two `\n` separated words
    - output: longest common subsequence
2. **Example operation**  
  `--- cormen/15-lcs ‹master* M?› » ./1-lcs`  
  `abrakadabra`  
  `yabbadabbadoo`  
  `abadaba`
